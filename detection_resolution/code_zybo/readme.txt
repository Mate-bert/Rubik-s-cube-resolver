C’est ce dossier que nous avons copié sur la Zybo pour exécuter notre code.
Par rapport à la version Windows, la syntaxe du code a été adaptée pour correspondre au compilateur disponible sur la Zybo.
Après avoir chargé le bitstream approprié pour activer la résolution ainsi que l’envoi via la BRAM, 
il suffit de lancer le script run.sh en utilisant la commande suivante depuis son répertoire :
./run.sh
